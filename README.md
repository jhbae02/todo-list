# ✅ TODO List

오늘 할 일을 간편하게 관리하는 순수 HTML/CSS/JS 기반의 할 일 관리 앱입니다.

## 미리보기

![TODO List App](https://github.com/jhbae02/todo-list/raw/main/preview.png)

## 주요 기능

- **할 일 추가** — 입력창에 텍스트 입력 후 `추가` 버튼 또는 `Enter` 키로 등록
- **완료 처리** — 체크박스 클릭 시 완료 표시(취소선) 및 완료 시간 기록
- **탭 필터** — 전체 / 진행 중 / 완료 탭으로 목록 구분 표시, 각 탭에 항목 수 뱃지 표시
- **전체 선택** — 진행 중 항목이 있을 때 상단 바에서 한 번에 전체 완료 처리
- **시간 기록** — 각 항목에 추가 시간 표시, 완료된 항목에는 완료 시간 추가 표시
- **완료 항목 삭제** — 하단 버튼으로 완료된 항목 일괄 삭제
- **남은 개수 표시** — 진행 중인 항목 수를 하단에 실시간 표시
- **빈 화면 안내** — 항목이 없을 때 일러스트와 안내 문구 표시

## 사용 방법

별도의 설치나 빌드 과정 없이 `index.html` 파일을 브라우저에서 바로 열어 사용할 수 있습니다.

```
todo-list/
├── index.html   # 구조
├── style.css    # 스타일
└── script.js    # 기능 로직
```

## 기술 스택

- HTML5
- CSS3
- JavaScript (Vanilla)
